名称来源于 东方 Project 中的角色 [幽谷响子(Kasodani Kyouko)](https://zh.moegirl.org.cn/%E5%B9%BD%E8%B0%B7%E5%93%8D%E5%AD%90)，~~虽然其实并不是复读机就是了~~。

目前部署于 QQ 1020056509，昵称为“門前の妖怪小娘”。
